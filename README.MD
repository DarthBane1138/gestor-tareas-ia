# Prueba Técnica Fullstack IA

## Descripción
Aplicación fullstack de gestión de tareas desarrollada como prueba técnica. El proyecto contempla frontend, backend, base de datos e integración de una funcionalidad basada en IA.

## Objetivo
Construir una plataforma funcional de tareas con persistencia, operaciones principales sobre tareas e integración de una funcionalidad de IA.

## Stack tecnológico
- Frontend: React
- Backend: Django / Django REST Framework
- Base de datos: PostgreSQL
- Contenedores: Docker Compose
- IA: LangChain

## Estructura del proyecto
- `frontend/`: aplicación cliente
- `backend/`: aplicación servidor
- `docs/`: documentación de apoyo
- `docker-compose.yml`: orquestación de servicios

## Estado del proyecto
Inicialización del repositorio y preparación de estructura base.

## Ejecución
Pendiente de implementación.

## Notas
Este repositorio corresponde a una solución desarrollada para una prueba técnica de tipo pair programming.
